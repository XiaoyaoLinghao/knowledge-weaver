# Knowledge Weaver

MCP Server for structured knowledge retrieval on top of DMA daily memory files.

## Overview

Knowledge Weaver provides an MCP (Model Context Protocol) server that enables structured retrieval of knowledge from DMA daily memory files. It supports semantic search, temporal queries, and topic-based navigation.

## Setup

```bash
pip install -e ".[dev]"
```

## Usage

```bash
python -m knowledge_weaver
```

## Development

```bash
pytest
```
